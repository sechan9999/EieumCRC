# 이음 민권센터 (Eieum Civil Rights Center)

> 조지아 애틀랜타 한인 및 아시아계 이민자 공동체의 사회적·인종적·경제적 정의를 위해 함께 합니다.

## 소개

이음 민권센터는 NAKASEC 네트워크 소속의 비영리 단체로, 조지아주 애틀랜타에서 한인 및 아시아계 미국인을 조직하여 이민자 권익을 옹호합니다. [Woori Center (PA)](https://www.wooricenterpa.org)의 모델을 참고하여 설립을 준비 중입니다.
